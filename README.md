# Instituto Glamour - Premium Website Redesign

Uma reformulação completa do site do Instituto Glamour, criando uma experiência digital premium que transmite autoridade, sofisticação, confiança e desejo.

## 🎨 Design

- **Estilo Premium**: Clean, luxuoso, minimalista, feminino sofisticado
- **Paleta de Cores**: Tons neutros sofisticados (off-white, bege suave, nude elegante, dourado discreto, cinza premium)
- **Tipografia**: Montserrat (sans-serif), PT Serif (serif), Great Vibes (display)
- **Animações**: Framer Motion para transições suaves e elegantes

## 🚀 Tecnologias

- **React 18** - Biblioteca JavaScript para construção de interfaces
- **Vite** - Build tool rápido e moderno
- **TailwindCSS** - Framework CSS utilitário
- **Framer Motion** - Biblioteca de animações
- **Lucide React** - Ícones modernos

## 📦 Instalação

```bash
npm install
```

## 🏃 Desenvolvimento

```bash
npm run dev
```

O site estará disponível em `http://localhost:5173`

## 🏗️ Build para Produção

```bash
npm run build
```
